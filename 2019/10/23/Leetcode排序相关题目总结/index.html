Leetcode 148：排序链表

题目描述

在 O(n log n) 时间复杂度和常数级空间复杂度下，对链表进行排序。

示例 1:

输入: 4->2->1->3
输出: 1->2->3->4
示例 2:

输入: -1->5->3->4->0
输出: -1->0->3->4->5

解题思路

1：比较直观的想法是将链表转化为数组，再排序，再重新构造链表

2：当然也可以实现基于链表的快速排序和归并排序。由于快排会用到大量的swap，而归并排序只是相邻元素间的copy,所以归并排序会更简单实现一点,效率也会更好。merge操作可以参考leetcode 21;

Leetcode 280:摆动排序

题目描述

给你一个无序的数组 nums, 将该数字 原地 重排后使得 nums[0] <= nums[1] >= nums[2] <= nums[3]…。

示例:

输入: nums = [3,5,2,1,6,4]
输出: 一个可能的解答是 [3,5,1,6,2,4]

题目思路：

  1. 先排序，然后从第二个元素开始，每两个元素交换一下位置。
  1. 第二种方法就像想我们能不能一步步的将数组按摆动的顺序排好？摆动的顺序用一个表达式来表示就是：当i为偶数时，nums[i] <= nums[i + 1];当i 为奇数时，nums[i]>= nums[i +1];如果实际顺序是相反的，通过一次交换就可以达到目的，并且不会和已排好序的部分冲突。

Leetcode 324: 摆动排序 II

题目描述

给定一个无序的数组 nums，将它重新排列成 nums[0] < nums[1] > nums[2] < nums[3]… 的顺序。

示例 1:

输入: nums = [1, 5, 1, 1, 6, 4]
输出: 一个可能的答案是 [1, 4, 1, 5, 1, 6]
示例 2:

输入: nums = [1, 3, 2, 2, 3, 1]
输出: 一个可能的答案是 [2, 3, 1, 3, 1, 2]

解题思路

相比于leetcode280 ,这题的要求更为严苛，没有了等于。所以之前的方法是不行了，不过我们还是可以先采用先排序再构造的思路。可以先将数组折半，然后再将后半段插入到前半段中。但这样是通不过[4，5， 5， 6]这个测试案例的。所以我们需要做出调整，将后半段倒着插入到倒着的前半段中，这样就可以了。

参考答案

  1. 上述算法的时间复杂度为O(nlogn),进阶的要求是时间复杂度为O(n),这个怎样做到呢？肯定是不能用排序了，但是我们又要保证交叉的顺序，topK算法正好可以达到要求。我们只需要用topk算法先将数组按小于median , 等于median,大于median,后续的算法就和上面一样了。

参考答案

如果需要使空间复杂度降到O(1),需要用到坐标映射。另外这个O(n) 的算法在leetcode上实际还没有O(nlogn)的算法快。

Leetcode 1122 :数组的相对排序

题目描述

给你两个数组，arr1 和 arr2，

arr2 中的元素各不相同
arr2 中的每个元素都出现在 arr1 中
对 arr1 中的元素进行排序，使 arr1 中项的相对顺序和 arr2 中的相对顺序相同。未在 arr2 中出现过的元素需要按照升序放在 arr1 的末尾。

示例：

输入：arr1 = [2,3,1,3,2,4,6,7,9,2,19], arr2 = [2,1,4,3,9,6]
输出：[2,2,2,1,4,3,3,9,6,7,19]

提示：

arr1.length, arr2.length <= 1000
0 <= arr1[i], arr2[i] <= 1000
arr2 中的元素 arr2[i] 各不相同
arr2 中的每个元素 arr2[i] 都出现在 arr1 中

解题思路

​ 可以用一个map<value, count>将arr1排序，由于arr1中的值的范围比较小，可以直接用一个数值代替map.先排arr2中的数，边排边从map中去掉。再排剩下的元素。

Leetcode 147:对链表进行插入排序

题目描述：

对链表进行插入排序。

插入排序的动画演示如上。从第一个元素开始，该链表可以被认为已经部分排序（用黑色表示）。
每次迭代时，从输入数据中移除一个元素（用红色表示），并原地将其插入到已排好序的链表中。

插入排序算法：

插入排序是迭代的，每次只移动一个元素，直到所有元素可以形成一个有序的输出列表。
每次迭代中，插入排序只从输入数据中移除一个待排序的元素，找到它在序列中适当的位置，并将其插入。
重复直到所有输入数据插入完为止。

解题思路

模仿数组的InsertSort,所要注意的是在head之前插入的问题和ListNode的next调整的顺序。
