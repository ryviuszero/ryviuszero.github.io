简介

停词，词干提取和词性还原

​ 停词（Stop words)是在文段中经常出现，当又不能显示出改文段与其他文段区别的词，一般会将其忽略．常用的包含停词表库有NLTK,spacy等，中文的可以在这个github仓库找到：https://github.com/goto456/stopwords．中文在分词上还有一个比较火的库叫:[结巴分词](https://github.com/fxsjy/jieba)

​ 词干提取(Stemming)和词性还原(Lemmatization)都是生成词根的方法，简单来说，词干提取就是通过去除前后缀来得到词干，而词性还原则是基于语法规则(或者说是词典)．实际运用的词干提取算法有Snowball,Lancaster．对于词性还原,在NLTK中使用的是wordNet.另外说一句，中文不需要进行词干提取和词性还原．

文档矩阵

​ 首先原始数据是一行的文本，target是多个标签．将所有的文本组成一个文本库，将文本中每个单词编号，得到一个字典，然后在统计每个文段中各个词出现的评论，便将一段文本转化为一个词频向量．

TF-IDF（词频-逆文件频率）

​ 如果一个词在所有的文段中都出现了，那么它就不太能代表这个文段的特点，如果一个词只在特定的文段出现，那么它就代表着这个文段的特点．

​ TF(term frequency) 表示一个词在文档中出现的频率：

​ IDF(inverse document frequency)是对单词提供多少信息的一种度量，等于log(语料库中文档的数量/术语出现过的文档的数量)．公式：

tfidf(t, d, D) = tf(t, d) *idf(t,D).详细解析可参考:TF-idf

SVD(奇异值分解)

​ 奇异值分解(Singular Value Decompoaition, SVD)就是将一个矩阵分解成两个酉矩阵（性质:A * A.T = eye)和一个对角阵，公式如下：

​ A=UΣV.T

​ SVD的作用：降维．因为在上述得到的矩阵维度高，十分稀疏．并且SVD 也可以得到协方差矩阵X.T * X 最大的 d 个特征向量张成的矩阵即：V,它可以用于主成分分词（PCA).

Truncated SVD, Randomized SVD(截断SVD 和随机ＳＶＤ)

​ 在SVD中是Amn = Um n * Σnm U(n*m).T,而在Truncated SVD,中则将右边的n换成了k,即只计算我们关注的k个topic,这样可以大大加快计算速度．Randomized SVD也是对SVD的一种优化，即保证结果差别不大的情况下，对速度和内存进行优化．
