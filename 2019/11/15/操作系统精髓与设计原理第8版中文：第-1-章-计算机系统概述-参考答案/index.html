操作系统精髓与设计原理第8版中文：第 1 章- 计算机系统概述 参考答案

发表于 2019-11-15

复习题

1.1 列出并简要地定义计算机的四个主要组成部分。

​ cpu,内存，I/O设备，主存储器。

cpu: 计算处理单元；存储器(内存和主存)：储存数据和程序。I/O设备:外界与计算机系统数据交互的设备。系统总线：负责各个模块间的通信。

1.2 定义处理器寄存器的两种主要类别。

​ 数据和地址寄存器：储存器地址寄存器（Memory Address Register 储存器缓冲寄存器(Memory buffer register) , 输入输出地址寄存器(I/O Address Register), 输入输出缓存寄存器(I/O buffer Register);控制寄存器：程序计数器(Program Counter)，IR（Instruction register).

1.3 一般而言，一条机器指令能指定的四种不同操作是什么？

处理器-存储器；处理器-I/O;数据处理；控制。

1.4 什么是中断？

​ cpu暂停正在执行的指令去执行其它指令的机制。

1.5 多中断的处理方式是什么？

1.先到先处理；2. 基于优先级的多中断处理机制。

1.6 内存层次的各个元素间的特征是什么？

​ 容量，速度，价格。

1.7 什么是高速缓冲存储器？

​ cache是基于局部性原理，用来降低cpu等待内存响应时间小容量的高速读取存储器，一般直接集成在cpu中。

1.8 多处理器系统和多核系统的区别是什么？

​ 多处理器系统：一快主板上多核cpu，多核系统：一块cpu中集成多个核。 ，多核CPU和多CPU的区别主要在于性能和成本。多核CPU性能最好，但成本最高；多CPU成本小，便宜，但性能相对较差。

1.9 空间局部性和时间局部性间的区别是什么？

​ 时间局部性：如果一个信息项正在被访问，那么在近期它很可能还会被再次访问。
这当然是正确的，用过的数据当然可能再次被用到。

​ 空间局部性：在最近的将来将用到的信息很可能与现在正在使用的信息在空间地址上是临近的。

1.10开发空间局部性和时间局部性的策略是什么？

​ 空间策略：选择合适的块大小 ，设计合适的将新块加入高速缓存的映射算法；

​ 时间策略：将高速缓存中最长时间未被访问的块置换出去，并制定好同步内存和高速缓存的写策略。
